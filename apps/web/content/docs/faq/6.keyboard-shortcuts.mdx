## Navigation

| Shortcut                                                     | Action                              |
| ------------------------------------------------------------ | ----------------------------------- |
| <kbd>⌘</kbd> + <kbd>1</kbd>, <kbd>2</kbd>, <kbd>3</kbd>, ... | Switch to specific outer tab        |
| <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> / <kbd>→</kbd>    | Navigate to next/previous outer tab |
| <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> / <kbd>→</kbd>    | Navigate to next/previous inner tab |
| <kbd>⌥</kbd> + <kbd>S</kbd>                                  | Switch to summary tab               |
| <kbd>⌥</kbd> + <kbd>M</kbd>                                  | Switch to memos tab                 |
| <kbd>⌥</kbd> + <kbd>T</kbd>                                  | Switch to transcript tab            |

## Sidebar & panels

| Shortcut                     | Action              |
| ---------------------------- | ------------------- |
| <kbd>⌘</kbd> + <kbd>\\</kbd> | Toggle Left Sidebar |
| <kbd>⌘</kbd> + <kbd>J</kbd>  | Toggle AI assistant |

## Notes & tabs

| Shortcut                                   | Action             |
| ------------------------------------------ | ------------------ |
| <kbd>⌘</kbd> + <kbd>N</kbd>                | Create new note    |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>N</kbd> | Create new note and start listening |
| <kbd>⌘</kbd> + <kbd>O</kbd>                | Open note          |
| <kbd>⌘</kbd> + <kbd>T</kbd>                | Open new tab       |
| <kbd>⌘</kbd> + <kbd>W</kbd>                | Close current tab  |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>T</kbd> | Restore closed tab |

## Quick access

| Shortcut                                   | Action                         |
| ------------------------------------------ | ------------------------------ |
| <kbd>⌘</kbd> + <kbd>K</kbd>                | Focus Searchbar                |
| <kbd>⌘</kbd> + <kbd>F</kbd>                | Search inside note or editor   |
| <kbd>⌘</kbd> + <kbd>H</kbd>                | Find and Replace in note       |
| <kbd>⌘</kbd> + <kbd>,</kbd>                | Open App Settings              |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>,</kbd> | Open AI Settings               |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>C</kbd> | Open Calendar                  |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>D</kbd> | Open Daily Notes               |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>F</kbd> | Open Advanced search           |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>L</kbd> | Open Folders                   |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>O</kbd> | Open Contacts                  |

## Audio playback

| Shortcut        | Action                                      |
| --------------- | ------------------------------------------- |
| <kbd>Space</kbd> | Play/pause audio in transcript view        |

## Editing

| Shortcut                    | Action                                      |
| --------------------------- | ------------------------------------------- |
| <kbd>⌘</kbd> + <kbd>Z</kbd> | Undo (also restores recently deleted notes) |

## Application

| Shortcut                                          | Action                                         |
| ------------------------------------------------- | ---------------------------------------------- |
| <kbd>⌘</kbd> + <kbd>Q</kbd> (hold)                | Show quit overlay with gauge visual             |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>Q</kbd>        | Quit Completely                                |
